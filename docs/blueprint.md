# **App Name**: TextVision

## Core Features:

- Image Upload: Drag-and-drop area for image uploads.
- Text Extraction: Automatically extract text from uploaded images using OCR technology.
- Text Display: Display extracted text in a text area below the image.
- Copy Text: Copy the extracted text to the clipboard.
- Text Correction: Use AI to correct commonly mis-recognized characters and suggest punctuation to the user as a tool

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) for a professional and calm feel.
- Background color: White (#FAFAFA) for a clean and modern look.
- Accent color: Dark gray (#4A4A4A) for contrast and readability.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Use simple, professional icons for upload and copy actions.
- Clean and well-organized layout with clear visual hierarchy.
- Subtle animation when extracting text to show the process.
